# 个性化追问表

> 被 SKILL.md Phase 0 Step 0.2.5 引用
> 赛道确认后，根据垂类特征追问个性化信息，让生成的流水线不是通用模板。

## 追问规则（按垂类动态触发）

| 垂类 | 个性化追问 |
|------|-----------| 
| **穿搭** | "你是什么身材类型？小个子/微胖/梨形/苹果形/H 形" |
| **穿搭** | "你偏好的语气？闺蜜推荐型/专业教练型/真实体验型" |
| **健身** | "你的健身目标？减脂/增肌/体态矫正/综合" |
| **健身** | "你出镜吗？真人示范/AI 演示/图文为主" |
| **美食** | "你擅长什么菜系？中餐/西餐/烘焙/减脂餐" |
| **宠物** | "你养什么宠物？猫/狗/其他" |
| **宠物** | "你的宠物叫什么名字？"（文案中用名字更亲切） |
| **职场** | "你的职业背景？互联网/金融/体制内/自由职业" |
| **数码** | "你主要测评什么？手机/耳机/智能家居/综合" |
| **家居** | "你擅长什么风格？现代简约/日式/北欧/复古" |
| **玄学** | "你擅长什么方向？八字/风水/紫微/综合" |
| **通用** | "你的人设/身份标签？（如'90 后新手妈妈''10 年程序员'）" |

## 追问原则

- 每个垂类最多问 2 个个性化问题
- 用户说"你帮我定"→ 用知识库默认值
- 个性化信息会写入生成的 style-guide.md，让文案有"人味"
