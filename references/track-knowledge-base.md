## 小红书 × 育儿

| 维度 | 规格 |
|------|------|
| **受众** | 25-35 岁新手妈妈，焦虑型搜索决策 |
| **爆款公式** | 痛点场景 + 具体方案 + 情感共鸣 |
| **文案风格** | 闺蜜口吻，真实体验，数据支撑，400-800 字 |
| **配图规范** | 3:4 竖图 1075×1440，日系 Q 版或实拍，温馨色调 |
| **生图策略** | Nano Banana 2 中转 API，日系 Q 版，1075x1440 |
| **发布频率** | 日更或隔日更 |
| **选题方向** | 月龄发育指南、辅食食谱、睡眠训练、早教游戏、母婴好物、避雷 |
| **变现路径** | 母婴带货、知识付费、品牌合作 |
| **标签库** | #育儿经验 #新手妈妈 #宝宝发育 #科学育儿 #辅食添加 #睡眠训练 |
| **文案禁忌** | 严禁人名、严禁医疗建议、涉及健康必须加免责声明 |
